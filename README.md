# Sandbox CodeX

> Next Generation AI Sandbox IDE

Sandbox CodeX là một AI Sandbox IDE thế hệ mới giúp lập trình viên xây dựng, chạy thử và triển khai ứng dụng ngay trên trình duyệt trong môi trường cô lập (Sandbox).
